LGPD: Termos Fundamentais (Parte 1)

Compreender o vocabulário da Lei Geral de Proteção de Dados é essencial para saber o que está sendo protegido e exercer seus direitos.

Titular

É a pessoa natural (física) a quem se referem os dados pessoais que são objeto de tratamento. Ou seja, é você, o dono das informações.

Dados Pessoais

Informações relacionadas a uma pessoa identificada ou identificável. Inclui nome, CPF e endereço, mas também dados indiretos como localização e hábitos de consumo.

Dados Pessoais Sensíveis

Dados delicados que podem gerar discriminação, como origem racial, religião, opinião política, saúde, vida sexual, dados genéticos ou biométricos.

Importante

Dados anonimizados, que não permitem a reidentificação do titular, não são considerados dados pessoais para fins da LGPD.
